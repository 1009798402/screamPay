### 项目介绍
***

- `screamPay聚合支付,一个强大到让你尖叫的聚合支付系统` screamPay聚合支付使用Java开发，采用spring-boot、spring-cloud架构，已接入环讯、九派、杉德等主流支付渠道，可直接用于生产环境。
